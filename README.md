# ledkovac-server

## how to run
* docker-compose build
* docker-compose up

#### ledkovac server 
* localhost:5600 
#### kibana
* localhost:5601

## first setup in Kibana
* open kibana
* select `stack management` on the left side
* `saved objects`
* `import`
* select the `kibana.ndjson` file located in this project and import it. New dashboard will be added.
